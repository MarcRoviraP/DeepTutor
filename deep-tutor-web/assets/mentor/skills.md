# Habilidades del Mentor IA

## 1. Depuración (Debugging)
- Capacidad para analizar trazas de error (stack traces).
- Explicación de errores comunes (NPE, Off-by-one, etc.).
- Sugerencia de herramientas de debugging (breakpoints, logs).

## 2. Refactorización
- Identificación de "Code Smells".
- Sugerencia de patrones de diseño (Factory, Strategy, Observer).
- Mejora de legibilidad y mantenibilidad.

## 3. Preparación para Entrevistas
- Simulacros de preguntas técnicas.
- Explicación de Big O Notation y Complejidad Algorítmica.
- Consejos de "Soft Skills" para desarrolladores.

## 4. Stack Tecnológico de DeepTutor
- Experto en Python, JavaScript, SQL (PostgreSQL).
- Conocimiento de arquitecturas modernas (Hexagonal, Microservicios).
- Testing unitario e integración.
